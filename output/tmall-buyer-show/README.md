# 天猫买家秀 6 套图

本目录保留 SVG 源图；PNG 图片可通过 `python scripts/generate_tmall_buyer_show_png.py` 本地生成，但不会提交到 Git。

基于用户提供的餐边柜参考制作：下柜高90公分、宽120公分，上柜高110公分、宽120公分；生成时保持黑色外框、木色抽屉/柜门、长虹玻璃、开放咖啡操作区等主要产品外观特征一致，搭配真实素人居家环境与不刻意精修的软装杂物。

- `01-small-apartment-morning.svg`：小户型餐厅早晨，窗帘、拖鞋、纸箱和早餐杯，素人随手拍。
- `02-family-dining-evening.svg`：家庭餐厅傍晚，餐桌边角、儿童水杯、购物袋，生活感。
- `03-rental-home-kitchen.svg`：出租屋开放厨房，冰箱贴、塑料收纳盒、围裙，不精致。
- `04-balcony-sideboard-day.svg`：靠阳台餐边区，绿植、晾衣架虚影、猫抓板，真实居家。
- `05-weekend-breakfast.svg`：周末早餐后，面包袋、咖啡杯、厨房纸、杂物稍乱。
- `06-night-warm-light.svg`：夜晚暖灯，外卖袋、钥匙、账单、半开的抽屉，买家秀。
